Args: charged_frag_types (typing.List[str], optional): fragment types with charge. Defaults to [ ‘b_z1’,‘b_z2’,‘y_z1’, ‘y_z2’ ]. precursor_mz_min (int, optional): Use this to clip precursor df. Attributes: precursor_df (pd.DataFrame): precursor dataframe. fragment_mz_df (pd.DataFrame): fragment m/z dataframe. fragment_intensity_df (pd.DataFrame): fragment intensity dataframe. charged_frag_types (list): same as charged_frag_types in Args. min_precursor_mz (float): same as precursor_mz_min in Args. max_precursor_mz (float): same as precursor_mz_max in Args. decoy (str): same as decoy in Args. key_numeric_columns (list of str): key numeric columns to be saved into library/precursor_df in the hdf file. Others will be saved into library/mod_seq_df instead.

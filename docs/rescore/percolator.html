Percolator

 Percolator (percolator_model:str='linear',
             percolator_backend:str='sklearn', cv_fold:int=1,
             iter_num:int=5, ms2_ppm:bool=True, ms2_tol:float=20.0,
             model_mgr:peptdeep.pretrained_models.ModelManager=None)

Percolator model. In parameter list, perc_settings is

perc_settings = peptdeep.settings.global_settings['percolator']
Type Default Details
percolator_model str linear machine learning
model type for rescoring, could be:
“linear”: logistic regression
“random_forest”: random forest
Defaults to perc_settings[‘percolator_model’].
percolator_backend str sklearn sklearn or pytorch.
Defaults to perc_settings[‘percolator_backend’]
cv_fold int 1 cross-validation fold.
Defaults to perc_settings[‘cv_fold’].
iter_num int 5 percolator iteration number.
Defaults to perc_settings[‘percolator_iter_num’].
ms2_ppm bool True is ms2 tolerance the ppm.
Defaults to perc_settings[‘ms2_ppm’].
ms2_tol float 20.0 ms2 tolerance.
Defaults to perc_settings[‘ms2_tol’].
model_mgr ModelManager None peptdeep.pretrained_model.ModelManager.
If None, self.model_mgr will be init by default (see peptdeep.pretrained_models.ModelManager).
Defaults to None.

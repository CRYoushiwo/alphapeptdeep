match_first_last_profile_mz

 match_first_last_profile_mz (spec_mzs:numpy.ndarray,
                              query_mzs:numpy.ndarray,
                              spec_mz_tols:numpy.ndarray)

Matching query masses against sorted MS2/spec profile masses, both first and last m/z values are returned.

Type Details
spec_mzs ndarray MS2 or spec mz values, 1-D float array
query_mzs ndarray query mz values, n-D float array
spec_mz_tols ndarray Da tolerance array, same shape as spec_mzs
Returns ndarray 2D np.ndarray of int32 with first and last matched index
for the query mz. The shape is the same as (len(query_mzs),2).
-1 means no peaks are matched for the query mz

Returns ndarray np.ndarray of int32, the shape is the same as query_mzs.

                    spec_mz_tols:numpy.ndarray)

query_mzs ndarray query mz values, n-D float array
spec_mz_tols ndarray Da tolerance array, same shape as spec_mzs
Returns ndarray np.ndarray of int32, the shape is the same as query_mzs.
-1 means no peaks are matched for the query mz

PepSpecMatch.match_ms2_centroid

 PepSpecMatch.match_ms2_centroid (psm_df:pandas.core.frame.DataFrame,
                                  ms2_file_dict:dict,
                                  ms2_file_type:str='alphapept', ppm=True,
                                  tol=20.0)

Matching PSM dataframe against the ms2 files in ms2_file_dict This method will store matched values as attributes: - self.psm_df - self.fragment_mz_df - self.matched_intensity_df - self.matched_mz_err_df

Type Default Details
psm_df DataFrame PSM dataframe
ms2_file_dict dict raw_name: ms2_file_path or ms_reader object
ms2_file_type str alphapept or ‘mgf’, or ‘thermo’
ppm bool True Defaults to True.
tol float 20.0 PPM units, defaults to 20.0.

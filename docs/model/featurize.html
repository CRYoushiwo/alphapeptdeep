parse_mod_feature

 parse_mod_feature (nAA:int, mod_names:List[str], mod_sites:List[int])

Get modification feature of a given peptide (len=nAA). Note that site=0 is for peptide N-term modification, site=1 is for peptide C-term modification, and 1<=site<=nAA is for residue modifications on the peptide.

Type Details
nAA int the lenght of the peptide sequence
mod_names typing.List[str] the modification names
mod_sites typing.List[int] the modification sites corresponding
to mod_names on the peptide
Returns ndarray 2-D feature array with shape (nAA+2,mod_feature_size)

get_ascii_indices

 get_ascii_indices (seq_array:Union[List,numpy.ndarray])

Convert peptide sequences into ASCII code array. The values are from 0 to 127. Zeros are padded into the N- and C-term for each sequence.

Type Details
seq_array typing.Union[typing.List, numpy.ndarray] list or 1-D array of sequences.
Returns ndarray 2-D np.int32 array with the shape
(len(seq_array), max seq length+2).
For the the sequence whose length is shorter than max seq length,
zeros are padded to the missing values.

get_batch_aa_indices

 get_batch_aa_indices (seq_array:Union[List,numpy.ndarray])

Convert peptide sequences into AA ID array. ID=0 is reserved for masking, so ID of ‘A’ is 1, ID of ‘B’ is 2, …, ID of ‘Z’ is 26 (maximum). Zeros are padded into the N- and C-term for each sequence.

Type Details
seq_array typing.Union[typing.List, numpy.ndarray] list or 1-D array of sequences with the same length
Returns ndarray 2-D np.int32 array with the shape
(len(seq_array), len(seq_array[0])+2). Zeros is padded into the
N- and C-term of each sequence, so the 1st-D is len(seq_array[0])+2.
